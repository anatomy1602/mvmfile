# MVM v0.3 计划：Markdown 全功能支持

> 目标：让 MVM 能够表达所有 Markdown 能表达的内容，并在此基础上提供更强的语义化能力。

---

## 一、核心设计理念：语义优先

### 1.1 为什么不是"维度设计"？

| ❌ 维度设计 | ✅ 语义设计 |
|-----------|-----------|
| 一维：列表 | list（有序/无序） |
| 二维：表格 | table（数据表/对比表） |
| 三维：时间线 | timeline（叙事时间线） |

**问题**：同样是"二维结构"，语义完全不同。

例子：
- 表格：`姓名 | 年龄` → 语义是"结构化数据"
- 网格：`[图片][文字][图片][文字]` → 语义是"布局"
- 问答：`Q: xxx\nA: xxx` → 语义是"交互"

**结论**：MVM 应该按"语义类型"来定义结构，而不是"维度"。

### 1.2 语义类型体系

MVM 的内容类型分为 6 大类：

| 类别 | 类型 | 说明 |
|------|------|------|
| **文本类** | paragraph | 段落 |
| | quote | 引用块 |
| | code | 代码块 |
| **列表类** | list | 列表（有序/无序） |
| **结构化数据类** | table | 表格（语义化） |
| | key_value | 键值对 |
| **布局类** | grid | 网格布局 |
| | columns | 多列布局 |
| **交互类** | quiz | 问答 |
| | tab | 标签页 |
| | accordion | 手风琴 |
| **媒体类** | image | 图片 |
| | video | 视频 |
| | audio | 音频 |

### 2.2 表格设计（重点）

#### 2.2.1 语义化表格类型

```mvm
===MODEL===
@type table
@version 3.0
@requires tree

columns: array<Column>
rows: array<Row>
caption: text?
sortable: bool?
filterable: bool?

@type Column
key: string
label: string
type: enum(text|number|date|boolean)
width: int?

@type Row
data: map<string, any>
highlight: bool?
```

#### 2.2.2 使用示例

```mvm
===MSG===
type: table

  ## key: name
    label: 姓名
    type: text
  ## key: age
    label: 年龄
    type: number
  ## key: role
    label: 身份
    type: text
rows:
  ## data:
      name: 张三
      age: 18
      role: 主角
  ## data:
      name: 李四
      age: 20
      role: 配角
```

## 四、技术实现

### 4.1 表格组件实现

#### 4.1.1 Schema 定义

```python
@dataclass
class Column:
    key: str
    label: str
    type: str = "text"
    width: int = None

@dataclass
class Row:
    data: Dict[str, Any]
    highlight: bool = False

@dataclass
class Table:
    columns: List[Column]
    rows: List[Row]
    caption: str = None
    sortable: bool = False
    filterable: bool = False
```

#### 4.1.2 Plugin 实现

```python
class TablePlugin(ComponentPlugin):
    name = "table"

    def render(self, section: DataObject, children: list) -> str:
        columns = section.get("columns", [])
        rows = section.get("rows", [])
        caption = section.get("caption", "")

        html = f'<div class="mvm-table">'
        if caption:
            html += f'<div class="mvm-table-caption">{caption}</div>'
        html += '<table>'
        html += '<thead><tr>'
        for col in columns:
            html += f'<th>{col.get("label")}</th>'
        html += '</tr></thead>'
        html += '<tbody>'
        for row in rows:
            highlight = row.get("highlight", False)
            html += f'<tr class="{\"highlight\" if highlight else \"\"}">'
            for col in columns:
                key = col.get("key")
                value = row.get("data", {}).get(key, "")
                html += f'<td>{value}</td>'
            html += '</tr>'
        html += '</tbody></table></div>'
        return html

    def get_css(self) -> str:
        return """
        .mvm-table {
            margin: 1em 0;
            overflow-x: auto;
        }
        .mvm-table table {
            width: 100%;
            border-collapse: collapse;
        }
        .mvm-table th, .mvm-table td {
            border: 1px solid #ddd;
            padding: 8px;
            text-align: left;
        }
        .mvm-table th {
            background-color: #f2f2f2;
        }
        .mvm-table tr.highlight {
            background-color: #fffacd;
        }
        """
```

### 4.2 列表组件实现

```python
class ListPlugin(ComponentPlugin):
    name = "list"

    def render(self, section: DataObject, children: list) -> str:
        ordered = section.get("ordered", False)
        items = section.get("items", [])

        tag = "ol" if ordered else "ul"
        html = f'<{tag} class="mvm-list">'

        for item in items:
            content = item.get("content", "")
            subitems = item.get("subitems", [])
            html += f'<li>{content}'
            if subitems:
                html += self._render_subitems(subitems, ordered)
            html += '</li>'

        html += f'</{tag}>'
        return html

    def _render_subitems(self, subitems: list, ordered: bool) -> str:
        tag = "ol" if ordered else "ul"
        html = f'<{tag} class="mvm-sublist">'
        for item in subitems:
            content = item.get("content", "")
            html += f'<li>{content}</li>'
        html += f'</{tag}>'
        return html

    def get_css(self) -> str:
        return """
        .mvm-list {
            margin: 0.5em 0;
            padding-left: 2em;
        }
        .mvm-sublist {
            margin: 0.25em 0;
            padding-left: 1.5em;
        }
        """
```

## 六、与 Markdown 的对比

| 功能 | Markdown | MVM | 优势 |
|------|----------|-----|------|
| 表格 | `| A \| B \|` | `type: table` | 语义化、可排序、可筛选 |
| 列表 | `- item` | `type: list` | 支持嵌套、可扩展 |
| 引用 | `> quote` | `type: quote` | 支持作者、来源 |
| 代码块 | \`\`\`python | `type: code` | 语义化、可扩展 |
